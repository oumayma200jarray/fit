PLACEZ VOS VIDÉOS D'EXERCICES ICI

Pour ajouter vos vidéos d'exercices :

1. Copiez vos fichiers vidéo dans ce dossier
2. Nommez votre vidéo principale "exercice.mp4"
3. Ou modifiez le nom dans le code (lib/accueil.dart, ligne 47)

Formats supportés :
- MP4 (recommandé)
- AVI
- MOV
- MKV

Exemples de noms de fichiers :
- exercice.mp4
- cardio.mp4
- musculation.mp4
- yoga.mp4
- stretching.mp4

Après avoir ajouté vos vidéos :
1. Redémarrez l'application
2. Ou exécutez : flutter clean && flutter pub get

Note : Les vidéos doivent être de taille raisonnable (< 100MB) pour de meilleures performances.
